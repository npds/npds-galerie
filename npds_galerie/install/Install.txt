#######################################################################################
##
## INSTALLATION AUTOMATIQUE (RECOMMANDEE)
## 1. Décompresser l'archive dans le dossier modules de votre site
## 2. Via l'administration du portail aller dans Gestion/Installation Modules
## 3. En face td-galerie, cliquer sur installer le module et suivez la procédure
##
## Attention: si td-galerie 1.4 était installé auparavant sur votre site et que vous souhaitez
## conserver vos tables tdgal_*, veillez sauter l'étape de mise à jour de la base de données MYSQL
##
##
## INSTALLATION MANUELLE
## 1. Décompresser l'archive dans le dossier modules de votre site
## 2. Créer la base de données à l'aide du fichier SQL joint
## 3. Mettez un CHMOD 766 sur le fichier gal_conf.php
## 4. Mettez un CHMOD 777 sur les dossiers imgs/ et mini/
##
## CONFIGURATION
## Via Admin-Plugins de NPDS.
##
## UTILISATION DU MODULE
## Appel grace à une URL du type : 
##		<a href=modules.php?ModPath=td-galerie&ModStart=gal>Mes galeries</a>
##
## ADMINISTRATION DU MODULE
## Via Admin-Plugins de NPDS. 
##
## BLOC IMGALEA - GALERIE PHOTO
## Ce bloc n'est disponible que si vous avez fait une installation automatique, 
## si vous avez fait une installation manuelle, insérer le script imgalea.php
## ou le script imgalea_priv.php dans un bloc en respectant la syntaxe suivante:
##
## include#modules/td-galerie/imgalea.php
## ou
## include#modules/td-galerie/imgalea_priv.php
##
## remarque: 
## imgalea_priv.php permet de faire apparaitre également dans le bloc les vignettes
## provenant de galeries photo membre ou privées (autorisées à un groupe NPDS), mais pour pouvoir consulter
## les galeries en question, il faut bien sûr être inscrit, faire partie du groupe et être connecté.
##
## imgealea_priv.php permet donc de donner un aperçu des galeries privées de manière à inciter
## les utilisateurs à s'enregistrer et se connecter.
